# 페이지 6

## 효과 분석

- **발표 텍스트:**
  > "계산된 PV 패널 면적을 토대로 발전량, 냉난방 부하를 분석하여 전기 사용량 감소 계산"
- **이미지 생성용 설명 (수정안 3차 - 기제작 단면도 통합형):**
  기존에 완성된 3페이지의 **측면 단면도(Cross-section) 구도**를 재활용하여 내용을 확장한 형태입니다. 화면 중앙의 BIPV 창문을 축으로 내/외부가 나뉩니다.
  - **왼쪽 (건물 외부)**: 내리쬐는 태양광을 받아 BIPV가 전기 에너지를 생성 구동하는 모습.
  - **오른쪽 (건물 내부)**: 업무 중인 내실자의 모습(3페이지 구조 유지)과 함께, 벽/천장에 **에어컨(냉난방기)**이 새롭게 추가 배치됩니다. 창문(BIPV)에서 갓 생산된 전기가 **에어컨으로 직접 직통해 들어가는 선명한 흐름선(배선)**이 그려집니다.
  - **효과 분석 시각화**: 실내 여백 한 쪽에, BIPV 도입으로 'Net 전기 사용량이 줄어들고 있음'을 부각하는 **데이터 대시보드(우하향 꺾은선 막대 그래프나 아래를 가리키는 큰 화살표)**가 배치됩니다.
