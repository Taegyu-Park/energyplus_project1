# 페이지 3 

## 최적화 개요
* **발표 텍스트:** 
  > "창호형 BIPV를 사용했을 때, 전기 발전+채광 조절 효과 얻을 수 있음. → 이 둘의 최적화 지점을 찾아보려 함."
* **이미지 생성용 설명 (수정안):**
  창호형 BIPV(건물일체형 태양광 발전)가 창문으로 설치된 건물의 **측면 단면도(Cross-section) 구도**. 화면 중앙에 위치한 BIPV 창문을 기준으로 좌우가 명확히 대비됩니다.
  - **왼쪽 (건물 외부)**: 내리쬐는 강한 태양광이 BIPV 패널에 비추어지고 있으며, 이로 인해 전기가 생산되는 모습(번개 모양이나 배터리 픽토그램 등)을 역동적으로 표현합니다.
  - **오른쪽 (건물 내부)**: 태양광이 BIPV를 통과하며 눈부시지 않도록 어느 정도 약화되고 정제된 부드러운 채광으로 공간을 밝히는 모습입니다. 내실자(사람)가 책상에 앉아 컴퓨터로 편안하게 업무를 보고 있는 모습을 담습니다.
