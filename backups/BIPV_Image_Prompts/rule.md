# BIPV 프로젝트 이미지 생성 규칙 (rule.md)

이 문서는 개별 `.md` 파일(예: `page_3.md`, `page_4.md` 등)을 토대로 발표용 이미지를 생성할 때 **반드시 적용해야 하는 가이드라인**입니다.

## 🔴 핵심 필수 규칙
1. **텍스트 영구 배제 (No Text / No typography)**
   - 이미지 내에 어떠한 알파벳, 숫자, 기호, 한글 등의 문리적 글씨도 포함되어서는 안 됩니다. 
   - 프롬프트 작성 시 네거티브 프롬프트(Negative Prompt)에 `text`, `letters`, `fonts`, `labels`, `numbers`, `watermark`, `words` 등을 반드시 강하게 추가합니다.
2. **깔끔한 발표용 자료 (Clean Presentation Style)**
   - 발표 자료(PPT)에 삽입했을 때 눈이 편안하고 가독성이 높은 디자인이어야 합니다.
   - 불필요하고 복잡한 배경은 배제하며 대상이 선명하게 보이도록 심플한 구도를 사용합니다.

## 🟢 프리미엄 퀄리티를 위한 추가 규칙 (권장)
3. **일관된 톤앤매너 (Consistent Color Palette)**
   - 친환경 에너지(EnergyPlus, BIPV)를 주제로 하는 만큼 '블루(Blue)', '그린(Green)', '화이트(White)' 등 신뢰감과 친환경성을 상징하는 특정 색상 팔레트 위주로 통일감을 줍니다.
4. **시각적 여백 확보 (Ample Negative Space)**
   - 이미지 생성 후 작성자가 PPT 화살표, 텍스트 상자 등을 덧입힐 수 있도록, 화면의 한쪽 면(좌/우 또는 상단)에 의도적으로 깔끔한 여백을 남겨놓습니다.
5. **2D 평면 아이콘 스타일 활용 (2D Flat Icon & Vector Graphics)**
   - 지나치게 사실적인 실사 사진이나 복잡한 3D 입체 그래픽보다는, 뱃지나 로고처럼 단정하고 일관된 느낌의 '2D 플랫 아이콘(Flat Icon)', '미니멀 2D 벡터 아트' 스타일을 강하게 권장합니다. (3D 그래픽/아이소메트릭 느낌 배제)
6. **16:9 와이드 비율 (Aspect Ratio 16:9)**
   - 일반적인 PPT(또는 발표 화면) 규격인 가로가 긴 비율(16:9 비율)을 기본 캔버스로 설정하여 레이아웃을 최적화합니다.
7. **빛과 흐름의 시각화 (Visualizing Flow & Light)**
   - '채광'과 '발전'의 최적화가 핵심이므로, 창문으로 들어오는 따뜻한 '햇빛의 입자'나 에너지가 전송되는 '에너지 흐름 선(Energy Flow lines)' 같은 효과를 그래픽적으로 추상화하여 화사하게 표현합니다.

## 📝 통합 적용 프로세스
- **Step 1:** 개별 `page_N.md` 파일에 적힌 **'이미지 생성용 설명'**을 읽고 시각적 대상을 파악합니다.
- **Step 2:** 본 문선의 **`核心 필수 규칙`**과 **`추가 규칙`**을 결합하여 완벽한 프롬프트를 구성합니다.
- **Step 3:** 생성된 이미지에 깨진 글씨(Text Artifact)가 들어갔거나 너무 복잡하다면, 규칙 1번과 2번을 강조하여 재수정합니다.
