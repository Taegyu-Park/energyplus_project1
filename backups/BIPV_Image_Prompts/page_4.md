# 페이지 4

## 변수 설정

- **발표 텍스트:**
  > "각 공간의 역할에 맞는 채광량을 산정해 PV 셀 밀도/투과율을 결정한다. 이후 EP의 PhotovoltaicPerformance:Simple 항목을 조절하여 PV 셀이 차지하는 면적 다변화"
-* **이미지 생성용 설명 (BIPV 미적용 비교군 - 수정안 7차):**
  건물의 3개 층을 보여주는 **측면 단면도(Cross-section) 구도**. 화면 좌측 상단에는 태양이 있으며, 건물 좌측 외벽 전반을 덮치고 있는 **거대한 반투명 빛의 면적(Area/Wedge)** 형태의 태양 복사(Solar Radiation)가 표현되어 있습니다.
  - **1층~3층 전체 건물 공통사항**: 기존에 있던 모든 **PV 셀 다그리드 패턴이 완전히 제거**된 맑고 투명한 일반 유리창입니다. 또한, 창 밖에 떠 있던 **배터리 아이콘들도 모두 삭제**되었습니다.
  - **전 층 실내 채광 (공통)**: 창문을 가리는 PV 셀이 아예 없으므로, 오피스나 헬스장 할 것 없이 1층 카페와 동일하게 **모든 층이 가장 눈부시고 밝은 최대치의 자연 채광**을 받으며 통일감 있게 환한 실내로 묘사됩니다.
